DragonProxy

Java Program
Open source
Discontinued

A proxy made to allow Minecraft: Bedrock Edition clients to connect to Minecraft: Java Edition servers.

I developed DragonProxy from 2019-2020. At it's peak it had 1000 GitHub stars and 100+ watchers on GitHub. The project inspired Geyser. Geyser was then acquired by CubeCraft Games and I joined their team.
